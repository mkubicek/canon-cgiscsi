## Windows Network Monitor / Connection Utility

Downloaded artifact:
`downloads/DR-C225W_II_NWMonitorV1.3.0.0_WCSetupToolV.1.3.0.0_forWindows.zip`

Extracted files:

```text
DRC225W/Setup.exe
DRC225W/Setup.ini
DRC225W/NetworkMonitor/setup.exe
```

`Setup.ini` identifies the large nested installer as the NetworkMonitor target:

```text
[General]
TargetDir=NetworkMonitor
TargetName=Setup.exe

[PreInstall]
UninstallMSIProductID={B7D7A443-ACD2-4D83-BE18-9CE42BF9465F}
```

Curated strings from `DRC225W/NetworkMonitor/setup.exe` showed an InstallShield
bootstrapper and signing/certificate material, but no scanner protocol strings:

```text
RunISMSISetup
RunISMSIMajorUpgradeRemoval
ISSetup.dll
ISSetupStream
C:\CodeBases\isdev\src\Runtime\MSI\Shared\Setup\Setup___Win32_Release_Prerequisites_Unicode\setupPreReq.pdb
InternetQueryDataAvailable
InternetCanonicalizeUrlW
```

Negative protocol sweep for the network-tool bootstrapper:

```text
cgiscsi: not found
cgi-bin: not found
POST / scan SCSI command templates: not found
SetWindow / Inquiry / RequestSense command strings: not found
```

PE resource enumeration found installer UI resources only. The large payload is
an `ISSetupStream` overlay beginning at file offset `0x165a00`, with a visible
UTF-16 record name `0x0410.ini` followed by high-entropy data. It was not run.
